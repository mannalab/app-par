# BêraPAR *APP* #

> Implementação do *APP* BêraPAr em Flutter

## Desenvolvimento ##

Para o ambiente de desenvolvimento foram utilizados:

- Flutter v1.20.3
- Dart v2.9.2
- Android SDK v30.0.2
- Android Studio v4.0
- OpenJDK v1.8.0


**Execução**
- Para executar o app utilize o comando ```flutter run```
